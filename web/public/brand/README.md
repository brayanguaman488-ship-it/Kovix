Coloca aqui los logos oficiales en PNG con fondo transparente:

- `kovix-logo-full.png` -> logo horizontal completo (escudo + texto)
- `kovix-logo-shield.png` -> solo escudo

Recomendado:
- Resolucion minima: 1200px de ancho para el logo completo.
- Exportar con fondo transparente (alpha), no gris.
- Mantener margenes limpios para que no se vea recortado.
